# Paws & Whiskers - Pet Grooming Website

This is a professional website for a pet grooming salon, created as part of a web development course project.

## Project Overview

**Business:** Paws & Whiskers Pet Grooming Salon  
**Owner:** Sarah Mitchell  
**Location:** Downtown, CA  
**Contact:** (555) 123-4567

## Features

- Responsive design that works on all devices
- Service showcase with pricing
- Before and after gallery
- Customer testimonials
- Online appointment booking form
- Contact form
- Smooth animations and transitions
- Mobile-first responsive layout

## Technology Stack

- HTML5
- CSS3 with animations
- Google Fonts
- Responsive Grid and Flexbox layouts

## Files Included

- `index.html` - Main website structure
- `styles.css` - Complete styling and responsive design
- `BUSINESS_SUMMARY.md` - Business overview document
- `INTERVIEW_TRANSCRIPT.md` - Interview with business owner
- `README.md` - This file

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. All pages load locally without requiring a server

## Design Decisions

- **Color Palette:** Warm oranges (#ff9a56, #ff7e5f), calming blue (#5b9aa0), cream background
- **Typography:** Poppins (body text), Playfair Display (headings)
- **Layout:** CSS Grid and Flexbox for responsive design
- **Animations:** Smooth transitions and hover effects for better UX
